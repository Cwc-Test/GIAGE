# What is GIAGE

G.I.A.G.E. (for GIAGE Is A Game Engine) is a multiplatform game engine project which use the [Sokol](https://github.com/floooh/sokol) library.

# Goals of GIAGE

-> Simple and fast
-> Multiplatform
-> Free
-> Usable by everybody !

You can contribute on this project with your knowledges.
